# Coding-interview

Algorithms implementations for coding interviews.

## Development

### Setup

```sh
pyenv install && poetry install
```

### Lint & Format

```sh
poetry run pysen run lint && poetry run pysen run format
```

### Test

```sh
poetry run pytest
```
